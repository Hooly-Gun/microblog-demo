## MicroBlog Demo

Prototyping a full Rails application (authentication, authorization, APIs, etc.)

### What does it do?

Features include:

* Multi-provider authentication using Devise and OmniAuth.
* Authorization using CanCan and Rolify.
* A RESTful API.
